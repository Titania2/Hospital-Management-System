# Hospital Management System (Qt 6.6.0)  
First Semester Project — Dec 2023

This project is a simple hospital management system built to make everyday hospital tasks easier and more organized. It provides a clean interface for both staff and doctors to manage patients, schedules, and internal communication.

## Features

### User Authentication  
- Secure sign-up and login for doctors and staff  

### Staff Management  
- Edit personal details  
- Schedule and manage patient appointments  
- Communicate within the system  

### Doctor Management  
- Manage availability and time slots  
- Access patient records and test results  
- Update patient information  

### Notes & Communication  
- Add and view notes for collaboration  
- Built-in messaging system for staff and doctors  

### Salary Management  
- Tracks working hours  
- Calculates salary based on logged time  

### Billing System  
- Generate patient bills based on services provided  

## Tech Stack  
- Qt Framework (6.6.0) – UI and application logic  
- SQL – database management
